\section{Experiments}

\subsection{Protocol}

We use the optimized Kalman filter as the primary baseline and compare it against image-conditioned residual models. Unless otherwise noted, models use a $400$ ms history window, an $800$ ms forecast horizon, normalized boxes, a batch size of $32$, AdamW training with learning rate $10^{-3}$, weight decay $10^{-4}$, Smooth L1 loss, gradient clipping at norm $10$, and validation-based checkpoint selection.

\subsection{Stage 1: Standard Split Without Decorrelation}

The first stage evaluates the canonical split without sample decorrelation. The baseline is the backpropagation-optimized Kalman filter. We then train residual models with representation combinations designed to isolate input families:

\begin{itemize}
  \item no image inputs, using Kalman/filter-state features only where configured;
  \item CSTR2 only and CSTR3 only;
  \item XT only, YT only, and XT+YT;
  \item event frames only;
  \item RGB or padded RGB only;
  \item event representations plus RGB;
  \item full representation sets such as XT+YT+CSTR3+RGB; and
  \item the same combinations with spatial cutouts around the anchor box.
\end{itemize}

\subsection{Implementation Details}

The default residual network uses one encoder per representation, concatenated pooled features, a fusion MLP, a history MLP over twelve box-time samples, and a residual MLP. Kalman trust parameters are initialized from the base configuration and optimized in log standard-deviation space. Evaluation reports learned model metrics, Kalman metrics with the \texttt{kalman\_} prefix, and last-four extrapolation metrics with the \texttt{last4\_} prefix.
